使用 matplotlib 读取图像并画框

2019年8月13日
| 计算机视觉
使用 matplotlib 读取图像，在图像上框出人脸部分 # 导入必要的库 import matplotlib.pyplot as plt import matplotlib.patches as patches from PIL import Image import numpy as np # 读取图像，转换为 numpy 数组 im = np.array(Image.open('arimura.jpg'), dtype=np.uint8) # 创建图像和轴 fig,ax = plt.subplots(1) ax.imshow(im) # 获取到面部位置，由 S3DF 模型获得 a = np.array([170, 88, 308, 301]) rect = patches.Rectangle((a[0],a[1]),a[2]-a[0],a[3]-a[1],linewidth=1,edgecolor='r',facecolor='none') ax.add_patch(rect) # 显示图像 plt.show() 结果如下：……
